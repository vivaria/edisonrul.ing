# Man Beast Of Ares

*   If “Man Beast of Ares” gains ATK, but then “Skill Drain” is activated, the ATK increase is wiped out. Even if “Skill Drain” is later destroyed, the ATK increase is not restored. (Although new ATK can be gained.)
