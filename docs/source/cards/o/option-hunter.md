# Option Hunter

*   If you send a monster to the Graveyard for “Option Hunter” that has undetermined ATK, you gain 0 Life Points. (“King of the Skull Servants”, “Megarock Dragon”, “Maju Garzett”, etc.)
*   “Option Hunter” does not target.
